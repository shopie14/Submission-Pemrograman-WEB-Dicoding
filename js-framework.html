JavaScript Framework

Framework Javascript memungkinkan seorang programmer untuk membuat halaman webnya menjadi responsif atau mobile-friendly. Responsif menjadi alasan lain pentingnya framework Javascript untuk programmer web. Karena pengguna internet hari ini kebanyakan menggunakan smartphone mereka untuk mengakses internet, maka fungsi responsif ini menjadi fitur yang tidak boleh ditinggalkan dalam proses pembuatan sebuah halaman web.

  1. AngularJS

    AngularJS adalah framework software open-source yang digunakan untuk membuat aplikasi single-page berbasis website. Framework ini juga cukup populer bagi para developer untuk membuat menu animasi di laman web HTML.

  2. React

    Sebenarnya React bukanlah sebuah framework, melainkan JavaScript library, tetapi banyak developer yang menganggapnya sebagai sebuah framework. Front end library ini dikembangkan oleh Facebook dan juga menggunakan bahasa yang bersifat open-source. React berfokus pada data rendering pada Document Object Model (DOM) yang menawarkan fungsi ideal untuk mengantisipasi traffic yang tinggi dan membutuhkan platform yang stabil untuk mengatasinya.

  3. Vue.js

    Vue.js adalah salah satu front end framework berbasis JavaScript yang paling populer saat ini. Platform ini unggul karena berukuran kecil dan menyediakan dua keuntungan utama, yakni Document Object Model (DOM) visual dan component-based. Vue merupakan framework yang sangat fleksibel untuk menambahkan dan mengurangi elemen pada situs Anda.

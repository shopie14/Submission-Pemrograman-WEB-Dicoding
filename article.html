HTML CSS

HTML (Hypertext Markup Languange) dan CSS (Cascade Style Sheet) adalah dua bahasa utama yang biasa digunakan untuk pengembangan website. Namun perlu diketahui HTML dan CSS bukan merupakan bahasa pemrograman. HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web, sementara CSS adalah bahasa styling yang digunakan untuk membuat halaman web lebih rapi dan menarik dengan menentukan warna, jenis huruf, tata letak dan berbagai aspek.

JavaScript

JavaScript adalah bahasa pemrograman web yang bersifat client side programming language, artinya kode program dieksekusi, diproses atau dijalankan di sisi client yang dalam hal ini web browser. JavaScript digunakan di sisi tampilan dengan membuat halaman web menjadi interaktif seperti pembuatan animasi, menangani event yang dilakukan user dan berbagai fungsi lainnya. Di dalam menggunakan JavaScript, 3 hal berikut merupakan hal-hal penting yang harus dipahami sebagai front-end developer:

DOM

DOM Manipulation DOM manipulation artinya memanipulasi dokumen atau elemen â€” elemen dalam HTML dengan DOM. DOM (Document Object Model) didefinisikan sebagai representasi elemen HTML pada dokumen menjadi sebuah object JavaScript. Karena elemen HTML pada suatu halaman direpresentasikan sebagai object JavaScript maka developer dapat melakukan perubahan atau manipulasi elemen HTML dengan menggunakan kode JavaScript. Manipulasi yang bisa dilakukan misalnya menambah animasi, membuat respon setiap terjadi event (misalnya klik), atau mengubah atribut elemen HTML.

AJAX / Fetch API

AJAX merupakan singkatan dari Asynchronous Javascript And XML. Fungsi Ajax ini secara umum adalah untuk pertukaran data secara Asynchronous (tidak langsung dan tanpa berpindah halaman). Sedangkan fetch adalah alternatif penggunaan AJAX yang mulai ada sejak versi ES6.

ES6+

ES6 yang biasa dikenal juga sebagai ES 2015 adalah sebuah standarisasi scripting language (Javascript) yang dibuat oleh European Computer Manufacturers Association (ECMA). Selain ES6 saat ini sudah ada versi ES7, ES8, dst. Versi ES6 ke atas ini biasa disebut ES6+. Pada setiap versinya banyak fitur baru maupun perbaikan dari versi sebelumnya. Sebagai calon front end developer, kamu perlu mengikuti standarisasi penulisan ES6+ ini.
